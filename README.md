# Safari School Management System

A comprehensive, production-ready school management system with role-based portals, data persistence, and AI-powered assistance.

## Features

- **Multi-role Portal**: Super Admin, Principal, Teacher, Student, Parent, Registrar, Finance, Librarian, Clinic, Counselor, Driver, Cafeteria, Alumni
- **Data Persistence**: All data saved to browser localStorage - no database setup required
- **AI Assistant**: Powered by Google Gemini but needs to be online and needs integration.
- **Public Website**: School information, admissions, announcements, tuition payments(leads to the login page)
- **Full CRUD Operations**: Create, read, update, delete for all modules
- **Real-time Updates**: Changes reflect immediately across all portals
- **Audit Logging**: All actions are tracked for security
